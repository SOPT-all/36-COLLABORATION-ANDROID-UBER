## 4팀의 프로젝트 기술 스택 선정 💚
🛠️ `클린아키텍처`
-> 기존 과제에서는 폴더링을 제대로 해보지 못해서 클린아키텍처를 공부하면서 폴더링을 해보고자 선정하게 되었습니다.

🛠️ `MVVM 패턴`
-> 합세의 규모가 그리 크지 않고 과제에서 MVVM 방식을 사용하여 더욱 익숙합니다. 러닝커브가 조금 높은 MVI를 적용하기보다 MVVM을 조금 더 체계적으로 사용해보고자 선정하였습니다.
